Commands:

(use a ; as a prefix for all commands)

Command Description Example
(search | s) <input> Search for <input> on youtube ;s abc will show youtube search results for "abc"
(choose | c) <input> Choose a song
Type a number to choose from the latest search results
Type a youtube song/playlist link to play a song/playlist		 ;c 2 will add the second song from the search results to the queue
leave Leave the voice channel
stop Stop playing the current song
queue Get the list of songs in the song queue
The latest queue message will auto-update
clear Clear the queue
play <number> Play <number> from the queue (will play the first song in the queue if no <number>)
next | skip Plays the next song in the queue
skipto <number> Skip to <number>
remove <number> Remove <number> from the queue
repeat Put the current song on/off repeat
restart Restart the current song
info Get info of <number> from the queue (will get current info if no <number>) ;info 4 will show the song info for the 4th song in the queue
pause Pause the song
resume | unpause Resume the song
move | reorder <number> Change position of a song in the queue ;move 5 3 will move the 5th song to the 3rd song in the queue
